# Utilities

CLICommand:  
A C# class that can handler the program startup parameters.  
  
CMarkup:  
A C++ XML parser.  
  
Junit:  
It can run Junit4 and Junit5 on the console mode.  
  
Tampermonkey:  
It hides the AD in the website, i.e, CSDN and Runoob.  

ThreadPool:  
The Java thread pool
